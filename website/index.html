Product System

Three practices. One coherent rhythm.

The system is built around three daily objects: a breath signal, a personal garden, and a quiet protocol. Each operates independently, yet together they form a recoverable daily pattern.

◐

Breath

A timed, phi-harmonic breathing signal. No instructions, no counting. Just a pulse at the golden-ratio interval that marks the boundary between doing and being.

✦

Garden

A minimal ritual interface for logging daily state. The garden grows not from streaks, but from the consistency of attention you bring to each entry.

○

Quiet

An ambient protocol that dims signal and deepens pause. Not distraction-blocking. Signal-reduction by design, implemented at the browser edge.

Deterministic Engine

Reproducible. Auditable. Not a black box.

The inference core is written in Rust and compiled to WebAssembly. Given the same input, it always returns the same output. Every decision path can be traced.

Same input. Same output. Always.

The engine runs entirely in the browser via WASM. No server-side inference. No model drift. The output depends only on the state vector you provide and the deterministic rules encoded in the binary.

We verify this with continuous equivalence testing: 10,000 random inputs fed to both the Rust reference and the WASM build. Any divergence fails the release gate.

Garden Flow

A ritual, not a routine.

The daily flow is designed to feel like tending something alive, not completing a checklist. Each step is optional, yet each step creates the conditions for the next.

1

Arrive

Open the garden. The interface shows only what is necessary: your state, the breath signal, and the space to log. No feeds. No notifications. No friction.

2

Signal

Follow the breath pulse for one cycle. The timing follows the golden ratio: inhale for 4 counts, hold for 2.4, exhale for 6.5. The body learns the rhythm without instruction.

3

Log

Record your state in the fewest possible dimensions. The engine uses this to compute tomorrow's suggested focus. The computation is local. The log is yours.

4

Rest

Close the garden. The quiet protocol remains active in the background, reducing ambient signal until you choose to return. No pressure to continue.
